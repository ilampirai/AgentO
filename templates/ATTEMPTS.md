# Attempted Actions

Log of failed actions and blocked patterns to prevent repeated failures.

## Blocked Patterns

Actions marked with `DONT_RETRY: true` will be blocked in future attempts.

### Format

```markdown
### [2024-01-15T10:30:00Z] npm install some-package
Error: Permission denied
DONT_RETRY: true
```
